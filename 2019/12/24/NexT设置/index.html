将 Hexo 部署到 GitHub Pages

github设置

New repository

新建一个名为你的用户名.github.io的仓库,例如我的wuyungen.github.io。

  1. 仓库名字必须是：username.github.io，其中username是你的用户名；
  2. 仓库创建成功不会立即生效，需要过一段时间，大概10-30分钟，或者更久，我的等了半个小时才生效；
  3. 每一个github账户最多只能创建一个这样可以直接使用域名访问的仓库
