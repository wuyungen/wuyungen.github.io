Homebrew入门到实战

发表于 2019-12-24

Homebrew 是使用 Mac OS 的一件利器，Homebrew 可以安装 Mac 没有预装但非常需要的东西。Homebrew 会将软件包安装到独立目录，并将其文件软链接至 /usr/local。

默认官方的更新源都是存放在 GitHub 上的，这也是中国大陆用户访问缓慢的原因，一般来说我们会更倾向选择国内提供的更新源，在此推荐中国科大、清华大学以及淘宝提供的更新源。
